Week 01 Tutorial Report – COIT20261 Network Services and Automation

This week’s tutorial focused on understanding the unit structure, setting up the required tools, and gaining basic hands-on experience with GNS3 and GitHub.

First, I reviewed the unit profile to understand the assessment requirements and overall expectations. I then ensured that all necessary software, including VirtualBox and GNS3, was installed and functioning correctly on my system. A private GitHub repository was also created following the required naming convention, and it was prepared for portfolio submissions.

Next, I created the Week 01 portfolio file in the repository and explored the use of Markdown for documentation. This helped in organizing content clearly and professionally, which is important for future tasks and collaboration.

In the practical section, I completed the GNS3 introductory task. I created a new project and added a Linux host node. I configured a static IP address by editing the /etc/network/interfaces file and verified the configuration using Linux commands in the console. I also added annotations such as project details and IP address within the network topology. The node was successfully started, and the IP address was displayed correctly.

Finally, I captured the required outputs, including the project export and screenshots of the network and console. This activity helped me understand basic network simulation, Linux configuration, and documentation practices.
